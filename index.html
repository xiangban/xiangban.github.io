圆角图片

发表于 2017-03-12

#####在开发过程中，经常能遇到圆角显示某一张图片，介绍几种常见的设置方式

  • 最简单，但是影响性能
    UIImageView *imageView = [[UIImageView alloc]initWithFrame:CGRectMake(100, 100, 100, 100)];
  imageView.layer.cornerRadius = imageView.frame.size.width / 2;
 //将多余的部分剪切
  imageView.layer.masksToBounds = YES;
  [self.view addSubview:imageView];
  • 使用贝塞尔曲线UIBezierPath和Core Graphics框架画出一个圆角
    UIImageView *imageView = [[UIImageView alloc]initWithFrame:CGRectMake(100, 100, 100, 100)];
  //开始对imageView进行画图
  UIGraphicsBeginImageContextWithOptions(imageView.bounds.size, NO, [UIScreen mainScreen].scale);
  //使用贝塞尔曲线画出一个圆形图
  [[UIBezierPath bezierPathWithRoundedRect:imageView.bounds cornerRadius:imageView.frame.size.width] addClip];
  [imageView drawRect:imageView.bounds];
  imageView.image = UIGraphicsGetImageFromCurrentImageContext();
   //结束画图
  UIGraphicsEndImageContext();
  [self.view addSubview:imageView];
  • 使用CAShapeLayer和UIBezierPath设置圆角
    导入
    UIImageView *imageView = [[UIImageView alloc]initWithFrame:CGRectMake(100, 100, 100, 100)];
  imageView.image = [UIImage imageNamed:@"1"];
  UIBezierPath *maskPath = [UIBezierPath bezierPathWithRoundedRect:imageView.bounds byRoundingCorners:UIRectCornerAllCorners cornerRadii:imageView.bounds.size];
  CAShapeLayer *maskLayer = [[CAShapeLayer alloc]init];
  //设置大小
  maskLayer.frame = imageView.bounds;
  //设置图形样子
  maskLayer.path = maskPath.CGPath;
  imageView.layer.mask = maskLayer;
  [self.view addSubview:imageView];

#####第三种最好，内存消耗少，渲染快

页面跳转的性能提升

发表于 2017-03-12

#####页面间跳转的性能问题大体可以分为以下三种

  • A页面跳转到B页面，在B界面需要加载大量的数据，导致页面跳转延迟
  • A页面跳转到B页面，在B界面需要加载大量UI元素，导致页面跳转延迟
  • A页面跳转到B页面，由于A页面或B页面的GPU使用率过高，所以导致页面跳转延迟

#####解决办法可以对应的分为三类

  • 利用多线程，将数据的获取放在子线程执行并返回结果，然后在主线程显示即可。
  • 在viewDidAppear中加载数据，跳转没延迟，但是显示会等待一段时间。也可以使用GCD。
  • 有种偷懒的方法就是，对视图控制器的图层做一次截图，把截图的结果设置为新图层的寄宿图
